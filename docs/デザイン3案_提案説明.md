# ホームページ デザイン 3案 — 提案説明

株式会社ヴィレッジ様のホームページについて、ご希望の印象（**信頼感・誠実・清潔感**）と、ご提示いただいた参考3サイトをもとに、方向性の異なる3案をご用意しました。
実際の御社情報（社名・キャッチコピー・5事業・強み・許可番号）を反映した仮デザインです。写真は差し替え前提の仮枠で表示しています。

## 案1｜信頼のネイビー
- **参考サイト**：株式会社ACTIVE（岡山）
- **イメージ**：堅実・力強い・プロフェッショナル。濃紺（ネイビー）を基調に、御社ロゴのVマークの青×緑をアクセントとして効かせ、「任せて安心」の信頼感を前面に出した王道デザイン。
- **配色**：ネイビー ＋ 空色／エメラルド ＋ 琥珀色のボタン（目に留まる差し色）
- **特徴**：ファーストビュー直下に「有資格者在籍・見積り無料・近隣対策・地域密着」を**訴求バッジ**として配置。事業内容はカード型で一覧性が高く、強みを数字（01〜04）で整理。BtoB・法人からの信頼を得やすい構成。
- **向いているケース**：品質・実績・専門性で選ばれたい。落ち着いた重厚感を出したい。

## 案2｜親しみのブルー×グリーン
- **参考サイト**：株式会社サンライズ（横浜市）
- **イメージ**：明るい・親しみやすい・地域密着。白基調に青と緑を効かせ、角丸と余白でやわらかく。個人のお客様（空き家・住宅解体・リフォーム）が「相談しやすい」と感じる親近感を重視。
- **配色**：ホワイト ＋ ブルー／グリーン ＋ オレンジのボタン、金色の実績メダル
- **特徴**：上部に電話番号と「かんたん無料見積り」を常時表示。ヒーロー横に**メダル型の安心バッジ**（有資格者・見積無料・近隣対策）。会社概要は見やすい表組み。問い合わせのハードルを下げる導線設計。
- **向いているケース**：個人客・地域集客を伸ばしたい。柔らかく親しみのある印象にしたい。

## 案3｜洗練のミニマル
- **参考サイト**：株式会社クリーンアイランド
- **イメージ**：清潔感・上質・スタイリッシュ。大きな英文コピーと広い余白、緑のスラッシュ（斜め線）をアクセントにした洗練デザイン。「解体業のイメージを変える」ような、他社と一線を画すブランド感。
- **配色**：近黒（ダーク）のヒーロー ＋ 白基調のセクション ＋ 緑のアクセント
- **特徴**：全面写真のヒーローに英語＋日本語の大見出し。事業内容は番号付きのリスト表示で読みやすく、情報を絞ってスマートに見せる。写真の力を最大限に活かす構成（＝良い写真素材が映える）。
- **向いているケース**：デザイン性・ブランドイメージを重視。若い層・法人にも上質さで訴求したい。

## 御社の状況をふまえた補足
- 参考3社は「累計◯千件・◯年」といった**実績数を前面に出しています**が、御社は**設立2024年**のため同じ土俵では戦えません。そのため3案とも、実績数の代わりに **①アスベスト調査の有資格者在籍（法令義務化された調査に自社対応可）②近隣対策の徹底・クレームの少なさ ③解体〜活用までのワンストップ ④見積り無料** を強み・訴求バッジとして据えています。
- どの案でも、**問い合わせフォーム・Googleマップ**は設置済みの想定です。
- 写真（現場・重機・施工前後・スタッフ・代表）が揃うほど、特に**案3**は完成度が大きく上がります。
